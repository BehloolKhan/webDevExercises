# webDevExercises
This repo will contain small web developement exercises I have done to enhance my understanding of html, css, javaScript and phpp

### How to run files:
The following contains intstruction to run the files in each the folders:
#### topic2-topic7:
##### opening any of the html files
* download each of the folders
* Open them up with VS code
* install the live server extension
* click on any of the html files
* click on the icon that says `Go live` at the bottom right of the html file
* Then, if you have microsoft edge, the rendered web page will pop up directly
* If not, then open up any web broswer and type in the following: `127.0.0.1:5500`
* It is import to note that the number **5500** is the port number.
* To see your port number, open vs code and open any of the html files.
* Where it normallly says, `Go live`, it should say **`Port: number`**
* The most common port numbers displayed are: **5500** and **5000**
* After you have typed in **`127.0.0.1:portNumber`**, you will see a display of all the css files/folders and html files and folder, click on the html file
* you want to see, alternatively, if inside a html folder, opn that folder and click on the file you want to see as a web page

##### closing the live server
If you dont want to see your html files, click on the bottom right of the vs code panel of where it says -> "Port: number"
This will close the live server and if you refresh your web page, you will not be able to see it anymore
